Создайте в Github новый репозиторий с именем git-branch

Инициализируйте локальный репозиторий с таким же именем

Создайте и переключитесь на новую ветку gitignore

Добавьте в проект .gitignore файл, как пример  https://github.com/aliaskov/bashscripts/blob/master/.gitignore 

Сделайте коммит и отправьте его с веткой gitignore в репозиторий git-branch


Пришлите ссылку на Ваш репозиторий с веткой gitignore


1.Содаем репозиторий в на гитхаб с файлом редми.
Копируем SSH код
2.Перейти в проводник в нужной папке правой кнопкой мыши нажать открыть гит баш здесь:
3.git clone git@github.com:MontyPy4/git-branch.git (Скопированный в гитхаб SSH)
4.ch git-branch
5.git checkout -b gitignore (создаем ветку и файл гитигнор)
6.vi .gitignore (создаем файл в текстовом редакторе и заходим в него. пишем что нужно и сохраняемся)
7.git status  (смотрим статус нашего файла-будет красный, так как мы еще не зафиксировани его и не добавили коммит)
8.git add .gitignore  (индексируем(фиксируем файл))
9.git commit -m "added gitignore"   (добавляем коммит)
10.git push origin gitignore   (пушим на удаленный репозиторий гитхаба фацл гитигнор. После он появится на гитхабе)
